# IPRS MVP (SecDB-like) Bootstrap

Generated by `bootstrap_mvp.py`.

## What you have
- `contracts/` JSON Schemas + fixtures
- `sql/` and `db/migrations/` Postgres DDL
- `openapi/openapi.yaml` API skeleton
- `compute/` worker + quantlib + pricers + golden tests

## Run tests
Create a venv and install:
- pytest
- psycopg[binary]

Then run:
  pytest -q

## Golden tests
Loan/Bond expected values are placeholders.
Run each test once, print outputs, then replace values in:
- compute/tests/golden/expected/*.json
